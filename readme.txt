Show a custom RSS feed with the latest posts from all your blogs

== Description ==

You can only use this plugin if you have a MU installation. The plugin settings can be found under Options-page.

== Installation ==

1. Unzip the file
2. Upload it to wp-content/plugins
3. Activate it from the plugins section
